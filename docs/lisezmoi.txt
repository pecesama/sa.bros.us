sabros.us 1.7

- Qu'est ce que sabros.us?

	sabros.us est un CMS (Système de gestion de contenu) vous permettant de gérer vos liens internet favoris, grâce à une interface en ligne et à une gestion des 'tags' (mots-clefs), comme le fait del.icio.us. Avec l'énorme différence cependant qu'ici vous avez le contrôle total sur le code source ! Ce dernier est écrit en PHP et supporte de multiples base de données, ce qui le rend indépendant de la plateforme sur laquelle il sera installé.

- Où puis-je le télécharger ?

	http://sourceforge.net/projects/sabrosus/

- Comment l'installer ?

        1.- Décompressez la distribution à la racine de votre site internet.
        2.- Lancez instalar.php dans votre navigateur.
        3.- Suivez les instructions.
	4.- Lancez exportar-ini.php  dans votre navigateur. 
        4.- Une fois terminé, effacez le fichier instalar.php

- D'autres données ?

       sabros.us est en cours de développement, et certains bugs sont encore en cours de correction. L'idée de diffuser sabros.us dans le domaine public vous autorise, vous encourage même, à lui apporter des améliorations, à le modifier, le tester... ou bien, si vous commencez à créer un CMS en partant de zéro, le code de sabros.us est une bonne base et il vous donnera une idée de la manière dont peut fonctionner un CMS.

- Qu'est ce que SopaSabrosa?

SopaSabrosa est un système pour organiser les données de tous les sabros.us. C'est le composant social de sabros.us!!!
